🧠 MODEL: HYBRID RL + PID CONTROLLER

A Deep Q-Network (DQN) Reinforcement Learning agent sits above a bank of PID loops. The RL agent learns the optimal pump-flow policy over time from real sensor data. PIDs handle fast-response stabilisation within each control cycle (10-second interval).

```
// RL AGENT — BELLMAN EQUATION
Q(s,a) ← Q(s,a) + α·[r + γ·max Q(s',a') − Q(s,a)]
// α=0.001 learning rate · γ=0.97 discount
// PID SUBORDINATE LOOP (each actuator)
u(t) = Kp·e(t) + Ki·∫e(t)dt + Kd·de/dt
// Kp=2.1, Ki=0.08, Kd=0.4 (tuned)
// REWARD FUNCTION (optimisation objective)
R = −w₁·(T_chip − T_target)² − w₂·P_pump − w₃·W_water
// w₁=10, w₂=0.5, w₃=0.3 (weighted)

📡 STATE VECTOR s ∈ ℝ¹²

T_chip (°C)T_coolant_inT_coolant_out T_ambientRH (%)Solar_irr (W/m²) Wind_spd (m/s)GPU_load (%)T_geo_loop PCM_stateTank_levelTime_of_day

⚡ ACTION SPACE a ∈ 𝒜 (7 actuators)

Pump_flow: LOW / MED / HIGH Wet_wall_valve: 0–100% PCM_bypass: ON / OFF Chimney_damper: 0–100% Geo_loop_valve: ON / OFF Recycle_pump: ON / OFF Alarm_flag: NORMAL / WARN / CRIT

🔁 DECISION LOGIC (per 10s cycle)

// RULE-BASED SAFETY WRAPPER (overrides RL)
IF T_chip > 38°C → a = PUMP_HIGH + WET_MAX
IF T_chip > 45°C → ALERT + emergency_override()
ELIF T_chip < 26°C → a = PUMP_LOW + PCM_STORE
ELSE → DQN_agent.predict(state_vector)

💧 IMMERSION COOLING CAPACITY

Q = ṁ · Cp · ΔT
// Mineral oil: Cp=1,670 J/kg·K
ṁ = Q / (Cp × ΔT)
= 106,400 / (1670 × 20)
= 3.19 kg/s ≈ 191 L/min

Pump power: P = ṁ·ΔP/ρ/η
≈ 1.8 kW (vs 35 kW chiller)
⟹ ΔT = 20°C confirmed at 191 L/min

🌍 GEOTHERMAL SINK CAPACITY

Q_geo = U · A · (T_fluid − T_earth)
// U=50 W/m²K · A=60m² · ΔT=8°C
Q_geo = 50 × 60 × 8
= 24,000 W = 24 kW

Remaining load to wet wall:
106.4 − 24 = 82.4 kW
⟹ 60m² PEX loop handles 22.6% of load

🎋 EVAPORATIVE WET WALL

Q_evap = ṁ_water × h_fg
// h_fg at 30°C = 2,431 kJ/kg
ṁ_water = Q / h_fg
= 82,400 / 2,431,000
= 0.034 kg/s ≈ 2.0 L/min

ΔT_air = 82,400/(ṁ_air × Cp_air)
≈ 6.8°C cooling
⟹ 2.0 L/min evaporation covers 82.4 kW

☀ SOLAR CHIMNEY DRAFT

v_chimney = √(2·g·H·(T_hot−T_amb)/T_amb)
// H=8m · ΔT=15°C · T_amb=303K
v = √(2×9.81×8×15/303)
= 2.81 m/s

Volume flow: Q_v = v × A_chimney
= 2.81 × 0.8 = 2.25 m³/s
⟹ 2.81 m/s passive exhaust, zero power

// L_f (paraffin) = 200 kJ/kg

= 12,600 kg ≈ 12.6 tonnes
